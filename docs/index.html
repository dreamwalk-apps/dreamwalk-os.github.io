Buttons

Collection of buttons common throughout DreamWalk projects

Block Button#

Common block button with support for loading/disabled states and additional styling.

Demo#

Usage#

import React, { useState } from "react";
import { StyleSheet, View } from "react-native";
import BlockButton from "../components/BlockButton";
export default function App() {
const [loading, setLoading] = useState(false);
const [disabled, setDisabled] = useState(false);
return (
<View style={styles.container}>
<BlockButton
text="Log in"
loading={loading}
disabled={disabled}
containerStyle={{ alignSelf: "stretch", marginTop: 20 }}
/>
</View>
);
}
const styles = StyleSheet.create({ ... });

Props#

text

Label value of the input

containerStyle

Style applied to the container pressable component

buttonStyle

Style applied to the button

bodyStyle

Style applied to the items within the button

textStyle

Style applied to the buttons label text component

textColor

Label text color

gradientColors

Colors to be used for the gradient of button for use with GradientBlockButton

gradientStart

Start position of gradient for use with GradientBlockButton

gradientEnd

End position of gradient for use with GradientBlockButton

loading

When true, triggers the loading indicator to display and also disables touches

disabed

When true, disables touch events on the button

disabed

Disable touch events on the button

gradientTransitionDuration

Length of transition time in ms when updating gradientColors prop

PostFixComponent

Component to render on right hand side of button

Events#

onPress

On press of button
