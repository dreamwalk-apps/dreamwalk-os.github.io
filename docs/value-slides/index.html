Value Slides

Collection of components for use with value slides.

Parallax slides#

Component that parallaxes images based on their current scroll position

Demo#

Usage#

To achieve this effect the images used in the slides need to be separated into a single background image, and any number of foreground images. The background image will remain static and move at the same speed as the scrolling/swiping. The foreground images will be positioned absolutely at their base position and then translate their X position based on the parameters passed. Slides must follow this model:

{
title: String,
description: String,
backgroundImage: Image.source,
overlayImages: [
{
uri: Image.source,
basePosition: {
top: Number?,
right: Number?,
bottom: Number?,
right: Number?,
},
translateMax: Number,
},
],
}
